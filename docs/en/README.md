# Welcome to Documentation Center

> A complete bilingual (Chinese / English) documentation system built with [Docsify](https://docsify.js.org).

## ✨ Features

- 🌐 **Bilingual** — Switch between Chinese and English instantly
- ⚡ **No build step** — Pure frontend rendering, deploy Markdown directly
- 🔍 **Full-text search** — Built-in search plugin
- 📱 **Responsive** — Mobile-friendly
