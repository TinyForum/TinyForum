Tiny Forum（天方论坛）是一个基于 MIT 协议开源的现代化论坛项目，提供了开箱即用的论坛解决方案。

!> 本项目正处于开发阶段，暂未正式发布。

## 前言

市面上有诸多论坛项目，但大多数项目都较为复杂，且不完全开源、建设不足，

例如 BBS-GO，这是促使本项目立项的直接导火索，该项目虽然开源，但前端现今已不开源，这可能是商业化、数据安全等方面的原因，暂且不论。但是确实让多数开发者和潜在合作伙伴望而却步。

因而，本项目以 MIT 全部源代码开源，需要说明的是，本项目由于立项以及初步实现事件极短，目前有很多功能并为实现，已经实现的也有很多部分都是 AI 加持开发的。业务功能，永远是本项目的主要目标。

Tiny Forum 是一个基于 Go 和 Next.js 的开源论坛项目，旨在提供一个简单、易用的社区平台。项目采用现代技术栈，包括 Gin、GORM、Next.js、PostgreSQL 等，为用户提供丰富的功能，包括帖子发布、评论、点赞、标签系统、用户管理等。
