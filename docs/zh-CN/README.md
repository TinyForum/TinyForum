# 天方论坛 - 简介

Tiny Forum（天方论坛）是一个基于 MIT 协议开源的现代化论坛项目，提供了开箱即用的论坛解决方案。

!> 本项目正处于开发阶段，暂未正式发布。

Tiny Forum 是一个基于 Go 和 Next.js 的开源论坛项目，旨在提供一个简单、易用的社区平台。项目采用现代技术栈，包括 Gin、GORM、Next.js、PostgreSQL 等，为用户提供丰富的功能，包括帖子发布、评论、点赞、标签系统、用户管理等。
