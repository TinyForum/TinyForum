# 介绍

本项目是一个基于 **Docsify** 的双语文档模板，支持中文和英文内容独立维护，通过 URL hash 前缀（`/zh-CN/` 和 `/en/`）区分语言版本。

## 工作原理

```
用户点击 [EN] 按钮
    ↓
location.hash 改为 #/en/...
    ↓
Docsify 根据 alias 规则加载 /en/ 目录下的 .md 文件
    ↓
侧边栏也来自 /en/_sidebar.md
```

## 为什么选择目录分离方式？

相比在同一个 Markdown 文件里用 HTML 注释切换语言，**目录分离**的优势在于：

- ✅ 每种语言独立维护，互不干扰
- ✅ 侧边栏结构可以按语言独立定制
- ✅ 搜索结果准确，不会混入另一语言的内容
- ✅ 便于团队分工翻译
